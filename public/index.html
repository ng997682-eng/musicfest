MusicFest 2026

Sistema web para gestionar artistas, presentaciones y boletos disponibles de un festival musical de tres días.

Inicio

Esta página sirve como base para construir una aplicación web. Incluye navegación por pestañas, consulta de datos desde el backend, formulario de registro, búsqueda y estadísticas.

¿Qué se puede modificar?

  • Los textos de cada sección.
  • Los campos del formulario.
  • El diseño visual en CSS.
  • Las rutas del backend.
  • Las funciones de JavaScript.

Artistas registrados

Agregar artista

Registra artistas, escenarios y boletos disponibles para el festival.

Buscar artista

Busca artistas por nombre, escenario o día de presentación.

Estadísticas del Festival

Consulta información general sobre artistas, boletos y presentaciones registradas.

Acerca del sistema

Este template usa HTML, CSS y JavaScript en el frontend. El backend está construido con Express y organiza la lógica en rutas, controladores y modelos.
