Acerca de TicketLive

Esta aplicación web permite gestionar conciertos independientes en distintas ciudades y recintos. El sistema permite registrar, consultar, actualizar y eliminar conciertos, además de buscar información y consultar estadísticas sobre precios y boletos disponibles.
