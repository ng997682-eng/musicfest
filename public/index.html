Catálogo de Productos

Template base para una aplicación web con frontend, backend y servicios REST.

Inicio

Esta página sirve como base para construir una aplicación web. Incluye navegación por pestañas, consulta de datos desde el backend, formulario de registro, búsqueda y estadísticas.

¿Qué se puede modificar?

  • Los textos de cada sección.
  • Los campos del formulario.
  • El diseño visual en CSS.
  • Las rutas del backend.
  • Las funciones de JavaScript.

Productos registrados

Consulta los productos guardados en el archivo productos.json.

Agregar producto

Captura la información del producto y envíala al backend.

Buscar producto

Busca productos por nombre o categoría.

Estadísticas

Calcula información general a partir de los productos registrados.

Acerca del sistema

Este template usa HTML, CSS y JavaScript en el frontend. El backend está construido con Express y organiza la lógica en rutas, controladores y modelos. 

        app.js
        controllers/productos.controller.js
        data/productos.json
        models/producto.model.js
        public/index.html
